# Preset: listicle

**Reads like**: Not Boring / Wait But Why / Stratechery "five things" pieces.
**Length**: 1200-2000 words.
**Voices**: none. Author voice forward.

## Structure

```
1. Title (H1, ≤70 chars) — names the count and the angle ("Six things the rare-earth halt is repricing", "Five EV makers that are most exposed").

2. Lede — one paragraph (80-130 words).
   - Names the framing (why this list, why now).
   - Sets the rubric (what the items have in common).
   - Telegraphs how the list is ordered (by severity? by likelihood? chronological?).

3. Numbered items — 5-12 items, each:
   - H2 header with the item title (e.g. "## 1. Mitsubishi Heavy was named by CSIS")
   - 2-3 paragraphs (180-280 words total).
   - Para 1: state the item + the concrete evidence {#source}.
   - Para 2: explain why this item belongs on the list (the rubric criterion).
   - Para 3 (optional): one actionable takeaway — what the reader watches or does.

4. Closing paragraph — one paragraph (80-130 words).
   - Pulls a pattern across the items (the meta-observation).
   - Names what to watch for next.
```

## Order matters

The first item is the strongest. The last item is the second-strongest (recency bias). The weakest items go in the middle. **Never end on a weak item** — it dominates the reader's takeaway.

## Header voice

The H2 item headers are themselves a list when read alone. Aim for parallel construction (all noun phrases, all complete sentences, all action verbs — but ONE consistent style across the list).

```
✓ Good (all complete factual statements):
## 1. Mitsubishi Heavy was named by CSIS
## 2. Eighty percent of EU firms sit within three hops
## 3. Less than 25% of export licenses are approved

✗ Bad (inconsistent):
## 1. Mitsubishi Heavy named (CSIS)
## 2. The 80% number — what it means
## 3. License approvals are tight
```

## Anti-patterns

- ❌ Listicles with 3 items pretending to be 3 articles → either commit to listicle or write op-ed.
- ❌ Item N+1 that repeats item 2 with a different example → de-dupe.
- ❌ Numbering that breaks parallel structure ("Three lessons" → only 2 are lessons, 1 is a question) → fix the framing.
- ❌ Ending with "What did I miss? Tell me on X." → that's a CTA, not a closer.
