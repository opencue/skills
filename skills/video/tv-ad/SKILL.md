# TV Ad

Create broadcast-quality TV advertisements with characters, narration, and VFX.

## When to activate

- User says "TV ad", "commercial", "brand video"
- User says "ad with character", "spokesperson video"
- User wants broadcast-quality advertising content
- User mentions "30-second spot", "15-second ad"

## Step 1 — Ad format selection

| Format | Duration | Structure |
|--------|----------|-----------|
| **Talking head** | 15-30s | Character speaks directly to camera |
| **Off-camera** | 15-60s | Narration over product/lifestyle shots |
| **VFX spectacle** | 15-30s | Character + impossible visuals |
| **Testimonial** | 30-60s | Real-feeling endorsement |

## Step 2 — Script structure (AIDA)

```
Attention (0-3s): Visual hook + opening line
Interest (3-10s): Problem statement or intrigue
Desire (10-20s): Solution showcase + benefits
Action (20-30s): CTA + brand lockup
```

## Step 3 — Character + Soul ID

- Create or select character via Soul ID
- Define wardrobe, setting, energy level
- Lock consistency for all shots in the ad

## Step 4 — Shot list with MCSLA

Generate each shot using cinematic formula:
- Opening: Wide/dramatic establishing shot
- Middle: Medium shots for dialogue, close-ups for emotion
- Product: Hero shot with perfect lighting
- End: Brand lockup with character or logo

## Step 5 — Delivery specs

```
📺 TV Ad Delivery:

  Broadcast: 1920×1080, 30fps, 15Mbps H.264
  Social cut: 1080×1920 (9:16), 1080×1080 (1:1)
  Audio: VO + music bed + SFX, -14 LUFS (broadcast standard)
  Supers: Legal text, CTA, URL
```

## Rules

- Always include brand lockup in final 2-3 seconds
- Audio must hit -14 LUFS for broadcast compliance
- Generate both landscape (TV) and portrait (social) cuts
- Character must be consistent across all shots (Soul ID)
- Include legal/disclaimer text placement in the shot list
