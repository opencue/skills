# Plugin snapshots — moved

Claude Code plugin snapshots (enabled set, marketplaces, plugin-bundled
MCPs) have been deduped and now live in:

→ **[recodeee/mcps](https://github.com/recodeee/mcps/tree/main/plugins)** — `plugins/plugins.sanitized.json`

Captures, per enabled plugin:
- name + marketplace
- latest cached version
- declared MCP servers (from the plugin's `.mcp.json` or
  `.claude-plugin/plugin.json`)
- enabled flag

Marketplaces from `~/.claude/settings.json` `extraKnownMarketplaces` are
captured alongside.

History note: `plugins/plugins.sanitized.json` was removed from this repo
on 2026-05-09 to dedup. Regenerated by
`recodeee/mcps/scripts/snapshot-mcps.py`.
